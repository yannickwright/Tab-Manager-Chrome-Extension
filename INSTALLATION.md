# Installation Guide

## Quick Start

Follow these steps to install and use your Tab Manager extension:

### 1. Load the Extension in Chrome

1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right corner)
4. Click **Load unpacked**
5. Select the `Tab Extension` folder
6. The extension should now appear in your extensions list

### 2. Pin the Extension (Optional but Recommended)

1. Click the puzzle piece icon (üß©) in your Chrome toolbar
2. Find "Tab Manager" in the list
3. Click the pin icon to pin it to your toolbar

### 3. Using the Extension

- Click the Tab Manager icon in your toolbar
- The extension will open in a new browser tab
- Click the `+` button to save your current tabs

### 4. Features Overview

#### Save Tabs

- Click the `+` button in the top right corner
- All your current tabs will be saved as a new session

#### Manage Sessions

- Click on a session name to expand/collapse it
- Click the pencil icon (‚úèÔ∏è) to rename a session
- Click individual tabs to reopen them
- Hover over tabs to see action buttons

#### Search

- Type to filter tabs by title or URL

#### Organize

- Use the "Expand/Collapse" button to toggle all sessions at once
- Delete individual tabs or entire sessions using the trash icon

## Troubleshooting

### Extension doesn't load

- Make sure all files are in the correct directory
- Check the Chrome console for errors (`chrome://extensions/` ‚Üí Details ‚Üí Errors)

### Icons not showing

- The extension includes placeholder icons
- You can replace them with custom icons in the `icons/` folder

### Tab Manager not opening

- Try reloading the extension
- Check if popup blockers are enabled

### Tabs not saving

- Check Chrome's storage permissions
- Open DevTools (F12) and check the Console for errors

## Updating the Extension

When you make changes to the code:

1. Go to `chrome://extensions/`
2. Find Tab Manager
3. Click the refresh icon (üîÑ)
4. The extension will reload with your changes

## Uninstalling

1. Go to `chrome://extensions/`
2. Find Tab Manager
3. Click "Remove"
4. Confirm the removal

## Need Help?

If you encounter any issues:

1. Check the browser console for errors
2. Make sure you're using the latest version of Chrome
3. Try removing and reinstalling the extension
